Linux部分常用命令
2017-07-29|Linux

最近在阿里云是买了个ECS服务器，装的Linux系统，自己搭建了一个java web 环境，其中通过百度学习了一些linux下的常用命令，记一下；

查看某个端口的占用情况

netstat -apn|grep 端口

查看tomcat的进程占用情况

ps -aux|grep tomcat

查看所有进程占用情况
​
​ netstat -apn

重启网络

service network restart

将1.repo重命名为1.repo.backup 常用作文件备份

mv /etc/1.repo /etc/1.repo.backup

查看网络信息

ifconfig

使配置生效

source /etc/profile

查看linux内在使用情况

free -m

清理内存

echo 1 > /proc/sys/vm/drop_caches

杀进程。pid 为相应的进程号

kill -9 pid

Mysql常用命令

启动Mysql

systemctl start mysqld.service

停止Mysql

tomcat 常用命令

以下命令在tomcat目录中的bin目录下执行：

./startup.sh         启动tomcat 
./shutdown.sh        停止tomcat

看tomcat的控制台输出

tail -f logs/catalina.out

看是否已经有tomcat在运行了

ps -ef |grep tomcat
